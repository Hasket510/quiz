# Quiz

Онлайн-тесты по вселенным Blizzard.  
Приложение позволяет выбрать мир, пройти серию вопросов с картинками, увидеть прогресс и получить результат.

## Демо

Проект задеплоен на GitHub Pages:

- **Продакшн**: https://hasket510.github.io/quiz/

## Возможности

- **Несколько вселенных Blizzard**  
  Тесты по мирам: Warcraft, Diablo, StarCraft.

- **Пошаговый квиз с прогресс-баром**
  - Вопросы идут по порядку, управление происходит через query-параметр `?question=...`.
  - Отображается «Вопрос X из N» и визуальный прогресс.

- **Картинки к каждому вопросу**  
  Для вопросов и результата используются настроенные изображения через компонент `Pic`.

- **Подсчёт результата**
  - Количество правильных ответов накапливается в `sessionStorage` (отдельно для каждого мира).
  - На странице результата корректный подсчёт с защитой от некорректных параметров в URL.

- **Маршрутизация по мирам**
  - Урлы вида:
    - `/:test?question=1` — страница игры для выбранного мира.
    - `/:test/result?correct=...&length=...` — страница результата.

## Технологии

- **React**
- **TypeScript**
- **React Router**
- **SCSS-модули**
- **Vite**

## Начало работы

### 1. Клонировать репозиторий

```bash
git clone https://github.com/Hasket510/quiz.git
cd quiz
```

### 2. Установить зависимости

```bash
npm install
```

### 3. Запуск dev-сервера

```bash
npm run dev
```

## Структура проекта

```text
src/
  App.tsx                 # маршрутизация приложения
  App.module.scss
  main.tsx                # точка входа
  styles/
    global.scss           # глобальные стили (reset + типографика)

  questions.ts            # типы и данные по мирам и вопросам

  components/
    Pic.tsx               # обёртка над изображениями вопросов/результатов

  pages/
    TestSelector/
      TestSelector.tsx
      TestSelector.module.scss

    Game/
      Game.tsx            # основная страница прохождения теста
      Game.module.scss

    Result/
      Result.tsx          # страница результата
      Result.module.scss

    NotFoundPage/
      NotFoundPage.tsx
      NotFoundPage.module.scss
```

## Идеи для развития

- **Таймер**
  - Ограничение по времени на каждый вопрос или на весь тест.
- **Локализация**
  - Добавить несколько языков (например, ru/en).
  - Хранить выбранный язык в `localStorage` или контексте.
- **Расширенная статистика**
  - Сохранять результаты в `localStorage`.
  - Показывать историю прохождений.
